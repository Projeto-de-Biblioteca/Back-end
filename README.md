# Biblioteca Digital - Back-end (API)

Repositório responsável pela lógica de negócio e persistência de dados da aplicação de biblioteca.

## Tecnologias Utilizadas
* **Node.js**: Ambiente de execução.
* **MongoDB**: Banco de dados NoSQL para persistência.
* **Mongoose**: ORM para modelagem e CRUD.

## Funcionalidades (CRUD)
* **Create**: Registro de novos títulos.
* **Read**: Listagem completa dos livros salvos.
* **Update**: Edição de informações existentes.
* **Delete**: Remoção de registros do banco de dados.

## Instalação
1. Clone o repositório.
2. Execute `npm install` para instalar as dependências.
3. Configure a string de conexão do MongoDB no arquivo `.env`.
4. Inicie o servidor com `npm start`.
